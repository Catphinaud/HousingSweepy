# HousingSweepy

## Feature Highlights
- Sweeps the housing wards automatically instead of going through each.

## Installation
- `https://raw.githubusercontent.com/Catphinaud/DalamudPlugins/refs/heads/main/pluginmaster.json`

## Usage
- `/sweepy` - Open the UI or just open the residental housing
- `/sweepy reset` - Reset the current state
